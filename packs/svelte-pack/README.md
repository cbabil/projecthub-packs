# Svelte Pack

Modern Svelte 5 + TypeScript workspace with Vite, ESLint, and Prettier.

## Contents

### Templates

- **svelte-gitignore**: Standard .gitignore for Svelte projects

### Workspaces

- **svelte-workspace**: Complete Svelte 5 + TypeScript setup with:
  - Svelte 5 with runes
  - Vite for fast development and building
  - TypeScript configuration
  - ESLint for linting
  - Prettier for code formatting

## Usage

1. Install this pack in ProjectHub
2. Create a new project using the svelte-workspace template
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier

## Requirements

- Node.js 18+
- npm or yarn
