# Node.js Pack

Modern Node.js + TypeScript workspace with Express, ESLint, and Prettier.

## Contents

### Templates

- **nodejs-gitignore**: Standard .gitignore for Node.js projects

### Workspaces

- **nodejs-workspace**: Complete Node.js + TypeScript setup with:
  - Express.js web server
  - TypeScript configuration
  - ESLint for linting
  - Prettier for code formatting

## Usage

1. Install this pack in ProjectHub
2. Create a new project using the nodejs-workspace template
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server

## Scripts

- `npm run dev` - Start development server with hot reload
- `npm run build` - Build for production
- `npm start` - Run production build
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier

## Requirements

- Node.js 18+
- npm or yarn
