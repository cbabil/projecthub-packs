# Python Pack

Python-focused assets for ProjectHub projects. Currently includes a .gitignore template; more Python starters can be added here.

## Contents
- `templates/python-gitignore`: Ignore patterns tailored for Python projects (`python.gitignore`).
- `templates/python-workspace`: Minimal Python workspace scaffold (`pyproject.toml`, `src/__init__.py`, `tests/__init__.py`).

## How to use
- For `.gitignore`: copy `templates/python-gitignore/python.gitignore` to your project root and rename to `.gitignore` (or append).
- For the workspace: copy `templates/python-workspace/` into your project, rename the project in `pyproject.toml`, and add dependencies.

## Compatibility
- ProjectHub >= 1.0
- Python projects (3.8+ recommended)
