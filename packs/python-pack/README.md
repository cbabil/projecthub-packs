# Python Pack

Python-focused assets for ProjectHub projects. Currently includes a .gitignore template; more Python starters can be added here.

## Contents
- `templates/python-gitignore`: Ignore patterns tailored for Python projects.
- `templates/python-workspace`: Default Python workspace layout (src/tests, pyproject).

## How to use
- For `.gitignore`: copy `templates/python-gitignore/python.gitignore` to your project root and rename to `.gitignore` (or append).
- For the workspace: copy `templates/python-workspace/` into your project, then rename the project in `pyproject.toml` and add dependencies.

## Compatibility
- ProjectHub >= 1.0
- Python projects (3.8+ recommended)
