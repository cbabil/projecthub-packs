# Rust Pack

Starter templates for Rust projects.

## Templates

### rust-gitignore
A comprehensive `.gitignore` file for Rust projects, covering:
- Build artifacts (`target/`)
- Cargo lock files (optional)
- IDE configurations
- OS-specific files

### rust-workspace
A complete Rust workspace setup including:
- `Cargo.toml` with workspace configuration
- Basic `src/main.rs` with hello world
- `.rustfmt.toml` for consistent formatting
- `.gitignore` for Rust projects

## Usage

1. Browse templates in ProjectHub
2. Select the template you need
3. Choose destination folder
4. Customize placeholders if prompted
